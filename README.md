# 🍽️ Restaurant Reservation System (MERN Stack)

A full-stack Restaurant Reservation System built using the MERN Stack (MongoDB, Express.js, React, Node.js). The application allows users to reserve restaurant tables online through a modern and responsive interface, while the backend securely manages reservation data using REST APIs and MongoDB.

Frontend Repository:
👉 https://github.com/karishma3005/ReserveBite_Frontend

Backend Repository:
👉 https://github.com/karishma3005/ReserveBite_Backend-

## 🚀 Features

- Online table reservation
- Responsive user interface
- Form validation
- Success confirmation page
- RESTful API integration
- MongoDB database storage
- Error handling
- Cross-Origin Resource Sharing (CORS) support

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Restaurant-Reservation-System.git
```

### 2. Navigate to the project

```bash
cd Restaurant-Reservation-System
```

### 3. Install Backend Dependencies

```bash
cd backend
npm install
```

### 4. Install Frontend Dependencies

```bash
cd ../frontend
npm install
```

## ▶️ Run the Project

### Start Backend

```bash
cd backend
npm start
```

### Start Frontend

```bash
cd frontend
npm run dev
```
